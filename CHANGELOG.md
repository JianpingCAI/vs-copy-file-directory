## [0.0.3] - 2024-01-XX

### Added

- Copy File Name command (without extension)
- Copy File Name with Extension command
- Copy File Path command (absolute path)
- Copy File Relative Path command
- Copy File Directory command (absolute directory)
- Copy Relative File Directory command
- Context menu integration for editor title
- Context menu integration for file explorer
- Keyboard shortcuts for main commands:
  - `Ctrl+Alt+F` / `Cmd+Alt+F` - Copy File Path
  - `Ctrl+Alt+P` / `Cmd+Alt+P` - Copy File Relative Path
  - `Ctrl+Alt+D` / `Cmd+Alt+D` - Copy File Directory
  - `Ctrl+Alt+R` / `Cmd+Alt+R` - Copy Relative File Directory
- Status bar feedback messages when content is copied
- Support for both active editor and file explorer context
